## Packages
recharts | Data visualization for the salary bar graph
framer-motion | Smooth page transitions and micro-interactions
react-webcam | Robust camera component for photo capture
lucide-react | Beautiful icons
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind CSS classes safely

## Notes
The application requires camera access for the photo capture feature.
Authentication is simulated via the provided API endpoint.
Salary strings (e.g., "$320,800") need parsing before graphing.
